# US-2.2 – Enforce Account Daily Limit

## Card

As a **Customer**, I **NEED** to be informed when my withdrawal amount exceeds the Account `dailyLimit` so that I can enter a valid amount.

## Conversation

- Before any funds check the ATM System validates the requested amount against `Account.dailyLimit`.
- If exceeded, the ATM displays a clear message and allows the Customer to enter a different amount.
- No Transaction is created in this case.
- Corresponds to activity diagram steps in [Withdraw Cash](../../use_case/withdraw_cash/withdrawCash.md): "Amount within daily limit?" → (no) → "Display 'Amount exceeds daily limit'".
- KANO: **Basic** — limit enforcement is a regulatory and contractual expectation.

## Confirmation

- **Given** the Customer has entered a withdrawal amount that exceeds the Account `dailyLimit`
  **When** the ATM System checks the amount
  **Then** the ATM displays "Amount exceeds daily limit" and no Transaction is created.
