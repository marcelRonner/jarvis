# US-2.3 – Verify Sufficient Account Balance

## Card

As a **Customer**, I **NEED** to be informed when my Account has insufficient balance so that I understand why the withdrawal was declined.

## Conversation

- After the daily limit check passes, the Bank Backend verifies that `Account.balance` ≥ requested amount.
- If insufficient, the ATM displays "Insufficient funds" and returns the Customer to the transaction selection screen.
- No Transaction is created.
- Corresponds to activity diagram steps in [Withdraw Cash](../../use_case/withdraw_cash/withdrawCash.md): "Sufficient balance on Account?" → (no) → "Display 'Insufficient funds'".
- KANO: **Basic** — customers expect a clear explanation when a withdrawal is declined.

## Confirmation

- **Given** the withdrawal amount is within the Account `dailyLimit` but exceeds the Account `balance`
  **When** the Bank Backend checks the balance
  **Then** the ATM displays "Insufficient funds" and no Transaction is created.
